Фаза 2: Тестирование @keyframes

Примечание: Эти анимации будут работать только после реализации runtime анимаций (Фаза 7). Сейчас тестируется только парсинг @keyframes правил.

Тест 1: fadeIn (from/to)

@keyframes fadeIn { from { opacity: 0; } to { opacity: 1; } }

Тест 2: slideIn (0%, 50%, 100%)

@keyframes slideIn { 0% { transform: translateX(-100px); } 50% { opacity: 0.5; } 100% { transform: translateX(0); } }

Тест 3: bounce (infinite)

@keyframes bounce { 0%, 100% { transform: translateY(0); } 50% { transform: translateY(-30px); } }

Тест 4: rotate (infinite)

@keyframes rotate { from { transform: rotate(0deg); } to { transform: rotate(360deg); } }

Тест 5: scale (infinite)

@keyframes scale { 0% { transform: scale(1); } 50% { transform: scale(1.5); } 100% { transform: scale(1); } }

Тест 6: colorChange (multiple colors)

@keyframes colorChange { 0% { background-color: #3498db; } 33% { background-color: #2ecc71; } 66% { background-color: #e74c3c; } 100% { background-color: #3498db; } }

Тест 7: complex (multiple properties)

@keyframes complex { 0% { transform: translateX(0) rotate(0deg); background-color: #3498db; } ... }
