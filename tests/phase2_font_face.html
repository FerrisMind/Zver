Фаза 2: Тестирование @font-face

Примечание: Шрифты должны загружаться из файла assets/fonts/Roboto-Regular.ttf. Если файл не найден, будет использован fallback шрифт.

Тест 1: Regular font-weight

@font-face { font-family: 'TestFont'; src: url('...') format('truetype'); font-weight: normal; }
The quick brown fox jumps over the lazy dog. 1234567890

Тест 2: Bold font-weight

@font-face { font-family: 'TestFont'; src: url('...') format('truetype'); font-weight: bold; }
The quick brown fox jumps over the lazy dog. 1234567890

Тест 3: Italic font-style

@font-face { font-family: 'TestFont'; src: url('...') format('truetype'); font-style: italic; }
The quick brown fox jumps over the lazy dog. 1234567890

Тест 4: local() fallback

@font-face { font-family: 'TestFontWithFallback'; src: local('Arial'), url('...') format('truetype'); }
The quick brown fox jumps over the lazy dog. 1234567890

Тест 5: font-display

@font-face { font-family: 'TestFontDisplay'; src: url('...') format('truetype'); font-display: swap; }
The quick brown fox jumps over the lazy dog. 1234567890

Тест 6: unicode-range

@font-face { font-family: 'TestFontUnicode'; src: url('...') format('truetype'); unicode-range: U+0020-007F; }
The quick brown fox jumps over the lazy dog. 1234567890
