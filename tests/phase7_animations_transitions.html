Фаза 7: Тестирование анимаций и transitions

Примечание: Эти тесты требуют runtime реализации анимаций и transitions. Transitions должны работать при изменении свойств (hover, focus, etc.). Animations должны запускаться автоматически при загрузке страницы.

Тест 1: Transition - цвет

transition: background-color 0.5s ease;
Наведите курсор (цвет должен плавно измениться)

Тест 2: Transition - transform

transition: transform 0.3s ease-in-out;
Наведите курсор (масштаб и поворот)

Тест 3: Transition - несколько свойств

transition: background-color 0.4s, transform 0.4s, border-radius 0.4s;
Наведите курсор (цвет, позиция, форма)

Тест 4: Transition - delay

transition: transform 0.5s ease 0.3s;
Наведите курсор (задержка 0.3s перед началом)

Тест 6: Animation - slide (infinite alternate)

animation: slide 2s ease-in-out infinite alternate;

Тест 7: Animation - rotate (infinite)

animation: rotate 2s linear infinite;

Тест 8: Animation - bounce

animation: bounce 1s ease-in-out infinite;

Тест 9: Animation - color change

animation: colorChange 3s ease-in-out infinite;

Тест 10: Animation - complex (multiple properties)

animation: complex 4s ease-in-out infinite;

Тест 11: Animation - duration

animation: slide 3s ease-in-out infinite;

Тест 12: Animation - delay

animation: slide 2s ease-in-out 1s infinite;

Тест 13: Animation - iteration count

animation: bounce 1s ease-in-out 3;

Анимация должна выполниться 3 раза и остановиться

Тест 14: Animation - direction (alternate-reverse)

animation: slide 2s ease-in-out infinite alternate-reverse;

Тест 15: Animation - fill-mode (forwards)

animation: slide 2s ease-in-out forwards;

Элемент должен остаться в конечной позиции после завершения
