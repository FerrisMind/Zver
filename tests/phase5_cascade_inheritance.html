Фаза 5: Тестирование каскада и наследования

Тест 1: Базовое наследование

.parent { color: #3498db; font-size: 20px; font-weight: bold; }
.child { /* наследует все от parent */ }
Родительский элемент
Дочерний элемент (наследует цвет, размер, жирность)

Тест 2: Переопределение наследуемого свойства

.parent { color: #3498db; }
.child-override { color: #e74c3c; }
Родительский элемент (синий)
Дочерний элемент (красный, переопределяет цвет)

Тест 3: Специфичность селекторов

.specificity-test { color: blue; } /* 0,0,1,0 */
div.specificity-test { color: green; } /* 0,0,1,1 */
#specificity-id { color: red; } /* 0,1,0,0 */
Класс (синий)
Класс + тег (зеленый)
ID (красный, самая высокая специфичность)

Тест 4: !important

.important-test { color: blue !important; }
.important-override { color: red; }
Этот текст должен быть синим (из-за !important), несмотря на более позднее правило

Тест 5: Inline стили

.inline-test { color: green; }
<div class="inline-test" style="color: orange;">
Этот текст должен быть оранжевым (inline стиль имеет более высокую специфичность)

Тест 6: Порядок правил (каскад)

.cascade-order-1 { color: red; }
.cascade-order-2 { color: blue; } /* Позднее правило побеждает */
Этот текст должен быть синим (позднее правило при одинаковой специфичности)

Тест 7: Наследуемые vs ненаследуемые свойства

.inheritance-parent {
  color: #2ecc71; /* наследуемое */
  background-color: #ecf0f1; /* ненаследуемое */
  border: 2px solid #3498db; /* ненаследуемое */
  padding: 20px; /* ненаследуемое */
}
Родительский элемент (зеленый текст, серый фон, синяя рамка, отступы)
Дочерний элемент (наследует зеленый цвет, НЕ наследует фон, рамку, отступы)

Тест 8: Множественное наследование

.grandparent { color: #9b59b6; font-size: 24px; }
.parent-inherit { color: #3498db; font-size: inherit; }
.child-inherit { /* наследует от parent */ }
Дедушка (фиолетовый, 24px)
Родитель (синий, 24px через inherit)
Ребенок (синий от parent, 24px от parent)

Тест 9: User-Agent стили

User-Agent стили должны применяться первыми (до author styles).

Например, элементы <h1> должны иметь больший размер шрифта по умолчанию.
