Фаза 3: Тестирование псевдоклассов

Инструкции:
  • Структурные псевдоклассы должны работать сразу после реализации
  • Псевдоклассы состояния (:hover, :focus, :active) требуют интеграции с UI событиями

Тест 1: :first-child

li:first-child { background-color: #3498db; }
  • Первый элемент (синий)
  • Второй элемент
  • Третий элемент

Тест 2: :last-child

li:last-child { background-color: #e74c3c; }
  • Первый элемент
  • Второй элемент
  • Последний элемент (красный)

Тест 3: :nth-child(2)

li:nth-child(2) { background-color: #2ecc71; }
  • Первый элемент
  • Второй элемент (зеленый)
  • Третий элемент

Тест 4: :nth-child(odd)

li:nth-child(odd) { background-color: #f39c12; }
  • Нечетный 1
  • Четный
  • Нечетный 2
  • Четный
  • Нечетный 3

Тест 5: :nth-child(even)

li:nth-child(even) { background-color: #9b59b6; }
  • Нечетный
  • Четный 1
  • Нечетный
  • Четный 2
  • Нечетный

Тест 6: :nth-child(3n)

li:nth-child(3n) { background-color: #1abc9c; }
  • Элемент 1
  • Элемент 2
  • Элемент 3 (выделен)
  • Элемент 4
  • Элемент 5
  • Элемент 6 (выделен)

Тест 7: :only-child

:only-child { background-color: #e67e22; }
Единственный дочерний элемент (оранжевый)

Тест 8: :first-of-type

p:first-of-type { background-color: #34495e; }

Первый параграф (темно-синий)

Div элемент

Второй параграф

Тест 9: :last-of-type

p:last-of-type { background-color: #16a085; }

Первый параграф

Div элемент

Последний параграф (бирюзовый)

Тест 10: :nth-of-type(2)

p:nth-of-type(2) { background-color: #c0392b; }

Первый параграф

Div элемент

Второй параграф (красный)

Третий параграф

Тест 11: :hover

.hover-test:hover { background-color: #3498db; }
Наведите курсор на этот блок (должен стать синим)

Тест 12: :focus

.focus-test:focus { border-color: #3498db; }

Тест 13: :active

.active-test:active { background-color: #e74c3c; }
Зажмите и удерживайте этот блок (должен стать красным)

Тест 14: :disabled

.disabled-test:disabled { background-color: #bdc3c7; }

Тест 15: :checked

.checkbox-test:checked + label { color: #2ecc71; }
