Добро пожаловать в Zver!

Экспериментальный браузерный движок на Rust

Возможности движка

Zver поддерживает основные веб-технологии:

  • HTML5 парсинг - полная поддержка современного HTML
  • CSS стили - базовые свойства и селекторы
  • JavaScript - выполнение JS кода через Boa
  • Layout движок - вычисление позиций элементов
  • GPU рендеринг - быстрый рендеринг через wgpu
